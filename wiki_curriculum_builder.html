📖 How This Works

Stage 1: Building the Candidate Pool

This tool uses Chinese Wikipedia as a content library to find reading material that can help you learn specific characters.

Setup Instructions

  1. Download Wikipedia dump:
    • XML bz2: Run python scripts/build_wiki_index.py --download
    • ZIM file: Download from Kiwix downloads
  2. Build the index:
    • For XML: python scripts/build_wiki_index.py --input dump.xml.bz2
    • For ZIM: python scripts/build_wiki_index.py --input wikipedia.zim (requires zim-tools or libzim)
  3. Copy to data folder - Move data/wiki_index.json to your web app's data/ folder.
  4. Load and query - Use this tool to find Wikipedia articles covering your target characters.

Algorithm

  1. Extract characters from your target text
  2. Subtract your known characters (from shared state)
  3. For each unknown character, find all Wikipedia pages containing it
  4. Count how many unknown chars each page covers
  5. Sort by coverage and return top candidates

Note: Future stages will add Jaccard clustering and path optimization.
