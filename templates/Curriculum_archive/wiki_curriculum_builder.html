📖 How This Works

Building the Candidate Pool

This tool uses a content library (Wikipedia, Classics, News, etc.) to find reading material that can help you learn specific characters.

Flask Server Setup

  1. Install Flask: pip install -r requirements.txt
  2. Prepare content sources:
    • Download and process Wikipedia dump: python scripts/build_wiki_index.py --input dump.xml.bz2 --extract-content --content-dir source/wiki/wiki_content
    • This creates:
      • source/wiki/wiki_data.json - Index file
      • source/wiki/wiki_content/ - Individual article files
  3. Start the server: python app.py
  4. Open in browser: Navigate to http://localhost:5000/wiki_curriculum_builder

Algorithm

  1. Extract characters from your target text
  2. Subtract your known characters (from shared state)
  3. Server searches the inverted index for pages containing unknown chars
  4. Count how many unknown chars each page covers
  5. Sort by coverage and return top candidates

Content Fetching

Click "Read" on any candidate to fetch the article content from the server. This only happens when you explicitly request it, keeping the workflow efficient.

Note: Future stages will add Jaccard clustering and path optimization.
