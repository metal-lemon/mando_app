      📖 How This Works

      Getting Started

      1. Go to "Can I Read This" and paste some Chinese text
      2. Click Analyze - this automatically saves the text and unknown characters to shared state
      3. Open Pathfinder - the target text loads automatically
      4. Click Build Learning Path to generate an ordered list of stories
      5. Follow the path to learn the characters you need!

      Algorithm

      Pathfinder finds stories that teach you the most target characters with the fewest new characters (efficient learning).
